## Description

This package provides a Deployer recipe to let you call Bugsnag ```php artisan bugsnag:deploy``` command when deploying.

## Install

Via Composer

``` bash
$ composer require tendersrl/bugsnag-deployer-recipe
```


## Minimum requirements

- Laravel 5.5
- PHP 7.0
